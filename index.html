about

lc-tools is a set of command line tools for managing clouds. It's based on libcloud and therefore supports all the providers that libcloud supports (EC2, Rackspace and GoGrid among of them). Like libcloud, it's written in Python.

lc-tools allows you to control your servers on the cloud, i.e. create, update, list, destroy, manage server images, query account special data like limits, IP addresses and so on. As of version 0.5.0.0 it supports managing load balancers as well.

Check examples and then documentation to get a better idea how lc-tools feels like.
