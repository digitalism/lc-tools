about

lc-tools is a set of command line tools for managing clouds. It's based on libcloud and therefore supports all the providers that libcloud supports (EC2, Rackspace and GoGrid among of them). Like libcloud, it's written in Python.

You might want to check posts in my blog with lc-tools tag with information about new lc-tools and libcloud features.

downloads and installation

lc-tools can be installed using PyPI:

sudo easy_install lctools

But it's recommended to use latest git version of lc-tools:

git://github.com/novel/lc-tools.git

It also needs svn version of libcloud, please check its developers info page to know how to get it.
